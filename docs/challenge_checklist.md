# Challenge Checklist — Verifiable Debate Arena (AI Judge)

## Public Eigen references used for requirement mapping

1. **Deterministic Execution** — EigenAI docs state same request/params/config should produce the same output bit-for-bit.  
   https://docs.eigencloud.xyz/eigenai/concepts/deterministic-execution
2. **Use EigenAI** — docs show seeded chat completions (`"seed": 42`) and grant-token onboarding.  
   https://docs.eigencloud.xyz/eigenai/howto/use-eigenai
3. **Verify Signature** — docs specify every response includes cryptographic signature and how to verify.  
   https://docs.eigencloud.xyz/eigenai/howto/verify-signature
4. **Use Cases** — includes **Verifiable AI Judges** as explicit EigenAI use case.  
   https://docs.eigencloud.xyz/eigenai/concepts/use-cases
