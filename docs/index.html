Verifiable Operator Copilot — Demo

Deterministic operator agent with signed run attestations for reproducible, multi-party trust.

What this demo proves

  • Deterministic run over Command Center snapshot
  • SHA-256 hashes for code + input + output
  • Signed run manifest with wallet identity
  • Independent signature verification

Run it yourself

git clone https://github.com/Halldon/command-center.git
cd command-center/eigen-verifiable-agent
./run_all.sh
