Verifiable Debate Arena (AI Judge)

Challenge-ready demo for EigenCloud: deterministic EigenAI judging, signed verdict artifacts, replayability, and sovereign-lite autonomous operation.

What this proves

  • Input = prompt + 2+ candidate arguments
  • Deterministic EigenAI judge scoring (fixed seed + model)
  • Signed verdict artifact (winner, per-debater scores, rubric, reasoning)
  • Verifiability bundle (input hash, seed, model, output hash, signatures, replay command)
  • Autonomous queue processing through existing watchdog loop

Quickstart

git clone https://github.com/Halldon/verifiable-operator-copilot.git
cd verifiable-operator-copilot
npm install

# one-command challenge validation (clean-state + replay checks + audit report)
npm run debate:demo

Manual run

# run one debate
npm run debate:run -- --input demo/debate_sample.json --seed 2026 --run-id demo-main

# verify cryptographic integrity of produced artifacts
npm run debate:verify -- --bundle artifacts/debate/runs/demo-main/verifiability_bundle.json

# replay and compare output hash against previous run
npm run debate:replay -- --bundle artifacts/debate/runs/demo-main/verifiability_bundle.json
